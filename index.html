Generative SVG with On-Demand Builders

A demonstration of returning a generative art SVG via Netlify On-Demand Builders

How to Use

Add any "seed" string after a slash on this site - /seed-word/ - to receive a generative art image! Due to how on-demand builders (ODB) work, each "seed" word will create an image that persists until the site build cache clears.

The default size is a 200px square. You can change the size by adding /seed-word/width/height/, optionally with only a width.

Be sure to ⭐️ Star on GitHub, and take it and improve it for your own projects!
